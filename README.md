# Investo

A full-stack investment management web application built with **React** for the frontend and **Django REST Framework** for the backend.

## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Django REST Framework  
- **Database:** PostgreSQL  
- **Version Control:** Git & GitHub
